STEP 1: INSTALL DEPENDENCIES (First Time Only)
═══════════════════════════════════════════════

   Windows Explorer → Double-click: INSTALL.bat
   
   This will:
   • Install Python packages (FastAPI, TensorFlow, etc.)
   • Install Node.js packages (React, Tailwind, etc.)
   • Create data directories
   
   Wait for: "Installation Complete!" message
   Time: ~5-10 minutes


STEP 2: START ALL SERVERS
═══════════════════════════

   Windows Explorer → Double-click: START_ALL.bat
   
   This will open 2 command windows:
   • Window 1: Backend Server (Port 8000)
   • Window 2: Frontend Server (Port 3000)
   
   Wait for both to show "Compiled successfully!" or "Application startup complete"
   Time: ~30-60 seconds


STEP 3: OPEN YOUR BROWSER
═══════════════════════════

   Open any web browser and go to:
   
   http://localhost:3000
   
   You should see the AirSense AI dashboard!

   1. Open Command Prompt
   2. cd c:\Users\Bala Muruganantham\Desktop\AIR_SENSE\backend
   3. pip install -r requirements.txt
   4. cd app
   5. python -m uvicorn main:app --reload

PROBLEM: Frontend won't start
SOLUTION:
   1. Open Command Prompt
   2. cd c:\Users\Bala Muruganantham\Desktop\AIR_SENSE\frontend\citizen-app
   3. npm install
   4. npm start

───────────────────────────────────────────────────────────────────────

PROBLEM: Port 3000 or 8000 already in use
SOLUTION:
   • Close other applications using these ports
   • Or change port in configuration files

───────────────────────────────────────────────────────────────────────

PROBLEM: Can't connect to API
SOLUTION:
   • Make sure backend is running first
   • Check http://localhost:8000/health
   • Check browser console for errors (F12)

───────────────────────────────────────────────────────────────────────

PROBLEM: Map not showing
SOLUTION:
   • Wait for page to fully load
   • Check browser console (F12)
   • Ensure internet connection (for map tiles)

───────────────────────────────────────────────────────────────────────

PROBLEM: Charts not displaying
SOLUTION:
   • Refresh the page
   • Clear browser cache
   • Check browser console (F12)
